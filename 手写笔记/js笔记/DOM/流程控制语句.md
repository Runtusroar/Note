## 一、条件判断语句
### 1、if语句
```js
//语法
//通常给语句加个代码块，使结构更清晰
if(条件表达式)
    语句...//一个语句或一个代码块

//执行流程
    先对条件表达式进行求值判断
        如果为true，则执行if后的语句
        如果为false，则不执行
```

### 2、if-else语句
```js
//语法
    if(条件表达式){
        语句...
    }else{
        语句...
    }

//执行流程
    先对条件表达式进行求值判断
        如果为ture，则执行if后面的语句
        如果为false，则执行else后的语句
```

### 3、if-else if-else语句
```js
//语法
    if(条件表达式){
        语句...
    }else if(条件表达式){
        语句...
    }else{
        语句...
    }
//实际就是 将else后的语句改为if-else语句
```
## 二、条件分支语句
### 1、swith语句
 ```js
//语法
    switch(表达式){
        case 表达式:
            代码...
        case 表达式:
            代码...
        case 表达式:
            代码...
        case 表达式:
            代码...
    }
//执行流程
    将switch表达式与case表达式依次比较
        如果为true，则执行该case后的所有代码
        如果为false，则往后比较，直至有true为止

//示例
    swith(1){
        case 1:
            console.log("壹")
        case 2:
            console.log("贰")
        case 3:
            console.log("叄")
    }
//输出 壹 贰 叄

//通常使用break来优化swith
    switch(表达式){
        case 表达式:
            代码...
            break
        case 表达式:
            代码...
            break
        case 表达式:
            代码...
            break
        case 表达式:
            代码...
            break
    }
```
## 三、循环语句
### 1、while语句
```js
//语法
    while(条件表达式){
        语句...
    }
//执行流程
    先对条件表达式进行判断
        如果为true，则执行一次循环体，再进行判断
            如果仍然为true，则再次执行循环体，后继续判断，如此重复
        如果为false，则循环结束

//一个循环通常 包含三个要件
    1、初始化表达式
    2、条件表达式
    3、更新表达式
```
### 2、do-while语句
```js
//语法
    do{
        语句...
    }while(条件表达式)
/执行流程
    先执行do后的循环体，再进行whlie后的条件判断
        如果为ture，则再次执行循环体，如此重复
        如果为false，循环终止

//while先判断再执行
//do-while先执行后判断
```
### 3、for语句
```js
//语法
    for(初始化表达式; 条件表达式; 更新表达式){
        语句...
    }
//执行流程
    1、先初始化表达式，再条件判断
    2、如果为true，则执行循环体
    3、执行更新表达式，再进行判断
    4、如果仍然为true，则执行循环体，如此重复
    5、如果为false，则终止循环

//三个表达式都可以省略
```

## 四、break和continue
### 1、break
```js
break
    用来终止switch和循环语句
    只会终止离他最近的循环
```
### 2、continue
```js
continue
    用来跳过当次循环
```
